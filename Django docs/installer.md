# UV, venv, and pip

## What is UV?
### UV is a new fast tool to install packages which helps to create virtual environment.

### it is a faster version of pip + venv

## what is Venv ?
### venv = virtual environment 

### it is a seperate folder where your project keeps its own python and packages.

## pip
### pip is a package installer which is used to download python libraries


## Conclusion
UV = fast pip + fast venv

venv = separate environment for each project

pip = installs Python packages
